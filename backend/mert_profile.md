## Backend
- ASP.NET Core Web API
- Katmanlı Mimari, Repository & Service Pattern

- Node.js & Express ile REST API geliştirme

## Frontend
- HTML5, CSS3, JavaScript (ES6)
- Angular ile component-based uygulamalar
- Responsive UI/UX prensipleri
- Kurumsal ekran tasarımı (dashboard, grid yapıları, form yönetimi)

## Veri & Altyapı Teknolojileri
- Oracle Database
- SQL Server
- Kafka & Hazelcast
- VoltDB tecrübesi (i2i Systems altyapıları üzerinden)
- Veri analizi, veri işleme, temel ETL mantığı

## Finansal Analiz & AI Çalışmaları
- Finansal zaman serisi verileri ile teknik analiz mantığı
- Grafik pattern analizi, destek/direnç yapısı, işlem senaryosu oluşturma
- AI tabanlı finansal model tasarımı üzerine araştırma:
  - POI tanıma
  - Breakout tespiti
  - Order block analizi
  - Haber akışı filtreleme & sentiment yaklaşımı
- Python ve API tabanlı finansal veri çekme & işleme mantığı
- n8n ile otomasyon akışları kurma (haber filtreleme, data pipeline)
